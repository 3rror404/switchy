Switchy

A tiny library for creating realistic, iOS style switches

Download

A Switchy switch is a checkbox-backed, draggable, swipeable and clickable switch that can be used as a form element.

Radio inputs can also be used.
Toggling requires JavaScript.

Text labels can be positioned before or after the switch.

Built in colour themes based on Apple's system colour scheme.

Built in helper classes for scaling individual switches.

Disabled switches.
